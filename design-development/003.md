# Console, CLI and SDK

- AWS Console, AWS CLI and AWS SDKs are the ways that you can interact and manage AWS services

- AWS SDK is the most usable for AWS developers

- Official SDKs
  - Java
  - .NET
  - C++
  - Python
  - Ruby
  - Go
  - Node.js

- Official ClientSide SDKs
  - JavaScript

- Official Mobile SDKs
  - Android
  - iOS
  - Xamarin
  - Unity
